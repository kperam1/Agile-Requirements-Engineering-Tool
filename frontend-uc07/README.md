# UC-07 Frontend (React + Vite)

Quick UI to test the Estimate Story endpoints.

## Run
```bash
npm i
npm run dev
# open the URL (normally http://localhost:5173)
```

- Backend must be running on `http://localhost:8080` or set `VITE_API_BASE`:
```bash
VITE_API_BASE=http://localhost:8090/api npm run dev
```

## Build
```bash
npm run build
npm run preview
```
